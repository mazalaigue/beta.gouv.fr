Des standards de qualité

Les services numériques construits dans le réseau beta.gouv.fr répondent à un haut niveau d'exigence :

  • Simplicité d'utilisation par une conception proche des utilisateurices
  • Accessibilité et conception pour tous
  • Protection des données personnelles
  • Écoconception et frugalité
  • Transparence via le code source et des statistiques ouverts
  • Sécurité agile
  • Qualité logicielle via code déployé et testé en continu
  • Qualité du support
